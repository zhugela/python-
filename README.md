## 学习阶段

### 第一阶段：基本手感
- 01 ~ 05：环境 + 基础语法 + 数据类型 + 流程控制 + 函数

### 第二阶段：能做事
- 06 ~ 11：文件操作 + 异常 + OOP + 模块 + 标准库 + 第三方库

### 第三阶段：项目能力
- 12 ~ 17：爬虫 + 数据分析 + Web开发 + 自动化 + 项目实战 + 刷题

## 推荐学习顺序

1. 先看 **01_环境安装** 配置好开发环境
2. 按顺序学 **02** → **05** 打好基础
3. 重点学 **06_文件操作**（后面自动化脚本全靠它）
4. 学 **12_爬虫入门** 做第一个项目：招聘信息采集系统
5. 用 **16_项目实战** 整合所学知识
